# Vite-Antd

## Project Setup

First, create a new project with the following command:

```bash
npm create vite@latest frontend -- --template react-ts
cd frontend
npm install
```

## Installing Dependencies

Install the necessary packages for the project:

### 1. **Ant Design** (UI Components)

```bash
npm install antd
npm install @ant-design/icons
```

### 2. **Chart.js** (For creating charts)

```bash
npm install chart.js react-chartjs-2
```

### 3. **React Router DOM** (For routing management)

```bash
npm install react-router-dom
npm install --save-dev @types/react-router-dom
```

## Usage

After installing the dependencies, you can start developing your project as needed:

```bash
npm run dev
```

Your web app will run at `http://localhost:5173/`
